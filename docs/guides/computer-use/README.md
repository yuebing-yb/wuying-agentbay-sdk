# Computer Use Guide

Computer Use is AgentBay's desktop automation environment for Windows and Linux systems.

These guides use Python for code examples — the concepts and API patterns apply to all supported languages (Python, TypeScript, Golang, Java).

> **💡 Async API Support**: This guide uses synchronous API. For async patterns, see [Async Agent API](../../../python/docs/api/async/async-agent.md).
