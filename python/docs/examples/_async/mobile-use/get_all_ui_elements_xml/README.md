## Get All UI Elements (XML)

This example demonstrates how to retrieve the full UI hierarchy in **XML** format.

### Prerequisites

- Python 3.8+
- `AGENTBAY_API_KEY` environment variable

### Run

```bash
export AGENTBAY_API_KEY=your_api_key_here
python main.py
```

### What you get

- `ui_dump.xml` saved in the current directory
- A short preview printed to stdout
