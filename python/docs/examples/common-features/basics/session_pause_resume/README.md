# Session Pause and Resume Example

This example demonstrates how to use the session pause and resume functionality in AgentBay SDK.

## Running the Example

```bash
cd python/docs/examples/common-features/basics/session_pause_resume
python main.py
```

## What This Example Demonstrates

1. **Basic Pause and Resume**: Simple workflow showing session pause and resume operations
2. **Session Status Monitoring**: Checking session status at different stages of the lifecycle
3. **Asynchronous Operations**: Using async variants for better performance
4. **Error Handling**: Handling common error scenarios gracefully

## Key Features

- Complete session lifecycle management (create → pause → resume → delete)
- Status verification at each stage
- Both synchronous and asynchronous operations
- Comprehensive error handling
- Clean resource cleanup

## Expected Output

The example will show:
- Session creation and ID
- Command execution results
- Pause/resume operation status
- Session status at different stages
- Error handling scenarios
- Cleanup confirmation
